## Uso

posicionado en la carpeta notes-app:

enlista todas las notas:
```
node app.js list
```

ingresar una nota:
```
node app.js add --title="otra nota" --body="nueva otra nota"
```


leer una nota:
```
node app.js read --title="nota"
```
